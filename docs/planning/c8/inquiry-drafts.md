## ドラフト1: PAY.JP営業/サポート宛(通常決済の先行開始+Platform利用照会)

### 注記(送信前に留意)
弁護士への正式相談(資金移動業/前払式支払手段の整理)は実施しない(受領8)。当該論点は下記質問⑥で貴社の一般的な整理を確認する形に一本化した。将来的にどうしても法的な確認が必要になった場合は、法テラス等の無料相談窓口で足りると判断している。

### 宛先の見つけ方
- 通常のお問い合わせフォーム: [pay.jp](https://pay.jp) サイト下部「お問い合わせ」、または [help.pay.jp](https://help.pay.jp) 内の問い合わせ導線。まずはここから通常決済(お支払いリンク)の利用開始を進める。
- Platform専用の申請・問い合わせ: [pay.jp/platform](https://pay.jp/platform) ページ内の申込・資料請求フォーム。Platform(Payouts型)のオプション導入について問い合わせるのはこちら。

### 文面

```
件名: PAY.JP通常決済のご利用開始+PAY.JP Platform(Payouts型)に関する事前照会

PAY.JP ご担当者様

突然のご連絡失礼いたします。個人事業主として、昆虫(クワガタ・カブトムシ等)の
飼育記録アプリと、飼育者間の個体売買マッチング機能を持つWebサービスを開発して
おります。売買はユーザー間(多くは一般の個人)で直接、銀行振込により成立させる
方式を既定としており、運営者である当方は代金そのものには一切関与しません。

運営者である当方は成約額の5%をシステム維持費として受け取りますが、これは
厳密な仲介ではなく、取引成立後に貴社の通常決済(お支払いリンク)でご請求する
緩やかな運用を予定しております。まずはこの5%システム維持費の請求手段として、
貴社の通常決済のご利用を先行して開始したいと考えております。あわせて、
将来的にユーザー間決済の選択肢を増やすため、貴社の「PAY.JP Platform」の
うちテナント(売主)が一般個人でも利用できるPayouts型を、取引ごとに選べる
オプション機能として導入することも検討しております。

契約前に、以下をご確認させていただけますでしょうか。

1. 個人事業主が運営するプラットフォームでも、Platform(Payouts型)のプラット
   フォーマー契約は可能でしょうか。
2. Payouts型テナント(売主・一般個人を想定)側に求められる本人確認書類・
   必要情報の登録実務を教えてください。
3. 通常の加盟店規約にある禁止商材規定「魚類・昆虫のうち適法に販売できるもの
   は販売を認める」という例外は、Platform(Payouts型)のテナント利用規約にも
   同様に適用されますでしょうか。
4. Platformプランの決済手数料率と、発生し得る月額固定費の見積り条件を
   教えてください。
5. 買主側の決済手段について、クレジットカード以外(コンビニ払い等)も
   利用可能でしょうか。
6. Platform(Payouts型)を利用した場合、プラットフォーマーである当方は資金を
   保持せず、貴社側で買主決済の集計・分配を行う構造になるかと存じますが、
   この場合プラットフォーマー側に資金移動業の登録は不要という理解でよいか、
   貴社の一般的なご整理をお伺いできますでしょうか。
7. テナントごとの最低振込額(minimum_transfer_amount)の設定可能な範囲と、
   振込手数料の負担条件について教えてください。

ご多用のところ恐縮ですが、ご回答いただけますと幸いです。何卒よろしく
お願いいたします。

[氏名]
[屋号(あれば)]
[メールアドレス]
```

## ドラフト2: PayPay for Developers/加盟店サポート宛(5%維持費請求の支払い手段追加・並行申請)

### 宛先の見つけ方
- [paypay.ne.jp/store-online](https://paypay.ne.jp/store-online/)(オンライン決済 for Developers)ページ内の申込・お問い合わせフォーム。
- 個別の技術・契約確認は同ページ経由、または加盟店サポート窓口(フォーム内で「契約に関するご相談」等を選択)。

### 文面

```
件名: PayPayオンライン決済(for Developers) ご利用に関する事前照会

PayPay株式会社 ご担当者様

個人事業主として、昆虫(クワガタ・カブトムシ等)の飼育記録・個体売買マッチング
Webサービスを開発しております。ユーザー間の物品売買は当事者間の銀行振込で
直接成立し、当方は一切関与しません。運営者である当方は成約額に対するシステム
維持費(5%)を、現在PAY.JPの通常決済(お支払いリンク)で請求する運用を進めて
おり、その支払い手段の選択肢を増やすため、貴社のオンライン決済(OPA)の並行
申請を検討しております。契約前に以下をご確認させていただけますでしょうか。

1. 実店舗を持たない、オンライン専業の個人事業主でも、オンライン決済(OPA)の
   加盟店契約は可能でしょうか。
2. 業態としては「マッチングプラットフォームのシステム利用料徴収」であり、
   物品そのものの売買はユーザー間の直接取引です。このような業態での
   申請は可能でしょうか。
3. 決済手数料率について、1.98%というご案内を目にしたことがある一方、
   物販向けには3.8%という情報も見ております。当方の利用形態にはどちらが
   適用されますでしょうか。
4. 審査に要する標準的な期間の目安を教えてください。

ご多用のところ恐縮ですが、ご回答いただけますと幸いです。何卒よろしく
お願いいたします。

[氏名]
[屋号(あれば)]
[メールアドレス]
```
